How to Use This Survey Tool

1. Purpose

This tool captures narrative evidence of Scripture use, access, and influence in a language community. It blends qualitative interviewing with a lightweight coding matrix for rapid situational assessment.

2. Workflow

  1. Build rapport (Warm Up) before deeper domains.
  2. Click a question: Live Guidance updates with probes.
  3. Record verbatim phrases & events; save when stable.
  4. Earn XP on first save; badges encourage thoroughness.
  5. Periodically run Auto Assess; refine flags manually.
  6. Assign Use Level (1â€“4) and a Primary Category.
  7. Export JSON for storage or analysis; Print summary if needed.

3. Live Guidance

Contextual microâ€‘prompts (Probe, Cue, Watch, Reminder) steer depth without leading. Section Tips summarize strategic aims. If no specific guidance, rely on core interview principles (neutral tone, ask for concrete recent stories, capture who / when / where).

4. Badges & XP

Gamification nudges completeness and richness:

  • +10 XP first save per question.
  • Badges (examples): Deep Narrative (long detailed account), Key Term Capture, Language Analyst, Story Weaver (varied sections), All Sections Complete.

5. Matrix Coding

Categories prioritize the dominant constraint or opportunity emerging from narratives:

  • T Translation: naturalness, clarity, key terms, trust.
  • SE Scripture Engagement: access channels, formats, acquisition, literacy.
  • S Spiritual Climate: openness, hunger, resistance social dynamics.
  • M Multilingual Dynamics: language choices, inclusion/exclusion, prestige.
  • NI No Issues: status quo adequate; low felt need for change.

Use Level (1â€“4) reflects depth of vernacular Scripture use: 1 = minimal / rare, 4 = integrated / normalized across domains.
